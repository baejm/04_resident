# 입주사 관리 프로그램


https://baejm.github.io/04_resident/




## 설명 

```
회사에서 처음으로 독학한 vue를 이용하여 만든 웹 주차관리 프로그램입니다.
WebSocket(웹소켓)을 이용하여 실시간으로 서버로 부터 정보를 주고 받아야 합니다.
※ 현재 서버는 이용하지 못해 임시로 ui만 살펴볼수 있습니다. 
```
